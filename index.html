ソフトウェアエンジニア | クラウド & CI/CD

要約

AWSクラウド環境を基盤としたWebアプリケーション開発を得意とするソフトウェアエンジニア。3年以上の経験を通じ、レガシーシステムの改善、CI/CDパイプラインの導入、サーバーレス技術の活用などに貢献。特に、システムのパフォーマンス最適化と開発プロセスの効率化において具体的な実績を上げてきました。技術力を通じて事業の成長に貢献します。

職務経歴

2024年9月 - 2025年8月

大手通信会社向けミドルウェアシステムの保守・機能拡張

社会インフラを支えるミドルウェアシステムの安定稼働を目的とした保守および機能拡張。15名チームのシステムエンジニア（保守・開発担当）として参画。

主な業務内容:

  • 既存システムの保守運用、および機能拡張の設計・実装
  • Jenkinsを用いたビルド・テスト・デプロイのパイプライン運用・改善
  • Shellスクリプトを活用したログ分析や障害調査

主な成果:

  • Jenkinsによるテスト自動化を拡大し、不具合発生率を50%削減。
  • デプロイプロセスを改善し、リリース作業時間を平均30%短縮。

2023年11月 - 2024年8月

倉庫輸送管理システム（WMS）刷新プロジェクト

旧式のWMSをコンテナベースのモダンなアーキテクチャへ移行。8名チームのシステムエンジニア（CI/CD担当）として参画。

主な業務内容:

  • Dockerを用いたアプリケーションのコンテナ化
  • Jenkinsを利用したCI/CDパイプラインの整備と自動化推進
  • 旧式SOAP APIからREST APIへの移行

主な成果:

  • CI/CD導入により、開発リードタイムを大幅に削減。
  • 大規模な出庫依頼時の処理時間を平均で40%短縮。

2023年4月 - 2023年10月

オンラインテストシステム

バックエンドをサーバーレスアーキテクチャで再構築。6名チームのクラウド・バックエンドエンジニアとして参画。

主な業務内容:

  • AWS Lambda, API Gateway, DynamoDBを中核とした設計・実装
  • 高トラフィックを想定したリアルタイム採点機能の実装
  • フロントエンド連携用のAPI仕様設計と実装

主な成果:

  • サーバーレス化により、インフラ維持コストを月額で大幅に削減。
  • 新機能のリリースサイクルを2週間から3日へ短縮。

2022年2月 - 2023年4月

AI教育プラットフォーム（AWS移行・開発）

オンプレミス環境からAWSへの全面移行と機能開発。13名チームのシステムエンジニア（AWS移行担当）として参画。

主な業務内容:

  • AWS（EC2, S3, RDS）へのインフラ移行プロジェクトを推進
  • CloudFrontによる教育動画コンテンツの低遅延化を実現
  • Dockerコンテナを用いたアプリケーション実行環境の設計・構築

主な成果:

  • AWS移行により、インフラのTCOを年間で約30%削減。
  • 海外ユーザーの動画再生開始時間を平均で50%以上短縮。

自己PR

私は、AWS環境での開発経験を活かし、システムの改善に貢献することにやりがいを感じるソフトウェアエンジニアです。単に仕様通りに実装するだけでなく、パフォーマンスや保守性といった非機能要件を常に意識し、より良いシステムを追求することを心がけています。

前職では、インフラのコード化（IaC）に携わり、手動作業によるミスを大幅に削減し、インフラの信頼性向上に貢献しました。また、サーバーレス技術を用いたプロジェクトでは、インフラコストと運用負荷の大幅な軽減を実現しました。

貴社においても、これまでのクラウド環境での開発経験と問題解決スキルを活かし、システムの品質と価値の向上に貢献できるものと確信しております。
